# How to Run

**Backend**

```sh
cd Backend/PIGGY
./gradlew bootRun
```



**Frontend**

```sh
cd Frontend/piggylife
yarn
yarn start
```



**AWS 주소**

```sh
http://piggylife.co.kr
```
